### C++服务器框架
---
### 编译环境
- `ubuntu 20.04`
- `cmake 3.16.3`
- `gcc 9.4.0`
- `yaml 0.6.2`
- `boost 1.71`

### 编译moka
- `mkdir build`
- `cd build`
- `cmake ..`
- `make`

### notice
- 源代码在编译时测试文件中的文件目录是绝对路径，需要更改
- 如需要在其他项目中使用该框架，需要在链接时链接到`/moka/lib/`中的动态库，然后在gcc中使用`-I`来查找对应头文件，既可使用相应的接口
